User Guide
Getting Started:
  1. Dataset Analysis:
    • Navigate to "Analyze Dataset" from the menu
    • Click "Analyze Dataset" button to process all comments
    • View results including sentiment distribution and word clouds
  2. Single Comment Analysis:
    • Go to "Single Comment" page
    • Enter your comment in the text area
    • Click "Analyze Comment" to see instant results
  3. View Results:
    • Access "Results" page for detailed analysis
    • Use search and filter options to explore data
    • Click on individual results for detailed views
Text Preprocessing
Preprocessing Steps:
Step Description Example
Cleaning Remove URLs, emails, punctuation, and numbers "Visit www.example.com!" → "Visit example com"
Normalization Convert text to lowercase "GREAT Product" → "great product"
Tokenization Split text into individual words "great product" → ["great", "product"]
Stop Word Removal Remove common words with no sentiment value ["the", "great", "product", "is"] → ["great", "product"]
Sentiment Classification
Classification Rules:
Sentiment Condition Icon Example
Positive Score > 0 "This is excellent and amazing" (Score: +2)
Negative Score < 0 "This is terrible and awful" (Score: -2)
Neutral Score = 0 "The product arrived on time" (Score: 0)
Score Calculation:
Formula: Sentiment Score = (Positive Word Count) - (Negative Word Count)

Example:
Comment: "This excellent product has poor quality"
Positive words: excellent (1)
Negative words: poor (1)
Score: 1 - 1 = 0 (Neutral)
API Documentation
Available API Endpoints:
POST /api/analyze

Description: Analyze a single comment

Request Body:

{
  "comment": "Your comment text here"
}

Response:

{
  "sentiment": "Positive",
  "score": 2,
  "positive_count": 2,
  "negative_count": 0,
  "original_text": "Your comment text here",
  "tokens": ["word1", "word2"]
}
GET /api/dataset-stats

Description: Get dataset statistics

Response:

{
  "total_comments": 40,
  "metadata": {
    "total_loaded": 40,
    "valid_comments": 40,
    "invalid_comments": 0,
    "source_file": "comments.csv"
  }
}
Limitations & Future Enhancements
Current Limitations:
  • Does not detect sarcasm or irony
  • Limited to English language
  • Context-independent word matching
  • Equal weight for all sentiment words
  • No emotion-level detection
Future Enhancements:
  • Multilingual support
  • Context-aware analysis
  • Sarcasm detection
  • Emotion classification
  • Real-time social media integration
  • Advanced machine learning models
